# Quickstart: 持仓页面

**Created**: 2026-04-04
**Feature**: 007-position-page

## 功能概述

持仓页面允许用户：
1. 查看当前持仓股票列表
2. 点击股票展开查看交易记录
3. 新增交易记录（买入/卖出/股息）
4. 编辑已有交易记录
5. 删除错误交易记录
6. 自动计算持仓均价和持仓数量

## 新增交易流程

1. 点击「新增交易」按钮
2. 填写表单：
   - 股票代码（如 `600519`）
   - 股票名称（如 `贵州茅台`）
   - 交易类型（买入/卖出/股息）
   - 交易日期
   - 交易价格
   - 交易数量
3. 点击保存
4. 系统自动计算持仓均价和持仓数量
5. 持仓列表自动更新

## 编辑交易流程

1. 点击股票展开交易记录列表
2. 点击交易记录的「编辑」按钮（✎）
3. 编辑表单显示原始值
4. 修改需要更正的信息
5. 点击保存
6. 直接更新数据库记录（不进行持仓计算）

## 删除交易流程

1. 点击股票展开交易记录列表
2. 点击交易记录的「删除」按钮（✕）
3. 确认删除提示
4. 记录从数据库中删除

## 交易计算规则

| 交易类型 | 持仓数量变化 | 持仓均价变化 |
|----------|-------------|-------------|
| 买入 | +交易数量 | 计入手续费后重新计算 |
| 卖出 | -交易数量 | 计入手续费和印花税后重新计算 |
| 股息 | 不变 | 减去股息价格 |

## 交易记录查询规则

- 只显示上次清仓（holding_count = 0）后的交易记录
- 如果从未清仓，显示所有历史记录
- 按交易日期倒序排列

## 数据存储

- 所有数据存储在 `trade_record` 表
- 字段包含：stock_code, stock_name, trade_date, trade_type, trade_price, trade_count, holding_count, holding_price
